# 日常高频词根

# cent

#### 相当于 on hundred 看到该词根联想到 ==百==

1. cent 	*n*. 美分

2. centigrade  *adj*. 摄氏的

   centi 百 + grade(级)

3. century  *n*. 世纪/百年

4. percent  *n*. 百分之...

   per (每一)+cent(百分)

5. percentage *adj*. 百分率

   per(每)+cent(百)+age

6. centimeter  *n*. 厘米

   centi(百)+meter(米)-->百分之一米 == 厘米

7. centennial   *adj*.百年纪念的 *n*.百年纪念

   cent(百)+enn(年)+ial
